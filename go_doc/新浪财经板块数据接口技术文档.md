## 概述

新浪财经提供了多个板块数据接口，支持获取概念板块、行业板块的资金流向、成分股等信息。本文档详细说明各接口的使用方法、参数、返回字段及Python实现示例。

### 基础信息
- **基础域名**: `vip.stock.finance.sina.com.cn`
- **API路径**: `/quotes_service/api/json_v2.php/`
- **数据格式**: JSON
- **请求方法**: GET
- **字符编码**: UTF-8

### 通用请求头
```python
headers = {
    "Host": "vip.stock.finance.sina.com.cn",
    "Referer": "https://finance.sina.com.cn",

}
```

## 概念板块资金排名接口

### 接口信息
- **接口名称**: 板块资金排名（概念板块）
- **接口路径**: `MoneyFlow.ssl_bkzj_bk`
- **完整URL**: `https://vip.stock.finance.sina.com.cn/quotes_service/api/json_v2.php/MoneyFlow.ssl_bkzj_bk`

### 请求参数

| 参数名 | 类型 | 必填 | 说明 | 示例值 |
|-------|------|------|------|--------|
| `page` | int | 是 | 页码，从1开始 | `1` |
| `num` | int | 是 | 每页数量，建议20-100 | `20` |
| `sort` | string | 是 | 排序字段 | `netamount` |
| `asc` | int | 是 | 排序方向，0=降序，1=升序 | `0` |
| `fenlei` | string | 是 | 分类：0=行业，1=概念，2=证监会行业 | `1` |
| `date` | string | 否 | 日期参数（但实际不返回历史数据） | `2025-11-13` |

### 排序字段说明（sort参数）

| 值 | 说明 |
|---|------|
| `netamount` | 净流入额（默认） |
| `inamount` | 流入资金 |
| `outamount` | 流出资金 |
| `ratioamount` | 净流入率 |
| `avg_changeratio` | 平均涨跌幅 |

### 返回字段说明

返回数据为数组格式，每个元素包含以下字段：

| 字段名 | 类型 | 说明 | 示例值 |
|-------|------|------|--------|
| `cate_type` | string | 分类类型 | `"1"` |
| `category` | string | 板块代码 | `"gn_ldc"` |
| `name` | string | 板块名称 | `"锂电池"` |
| `avg_price` | string | 板块平均价格 | `"21.0817"` |
| `avg_changeratio` | float | 板块平均涨跌幅 | `0.0415249` |
| `turnover` | string | 换手率 | `"586.822"` |
| `inamount` | string | 流入资金（元） | `"95751231452.9400"` |
| `outamount` | string | 流出资金（元） | `"71306794689.5100"` |
| `netamount` | string | 净流入额（元） | `"24444436763.4300"` |
| `ratioamount` | float | 净流入率 | `0.140198` |
| `ts_symbol` | string | 领涨股代码 | `"sz300037"` |
| `ts_name` | string | 领涨股名称 | `"新宙邦"` |
| `ts_trade` | string | 领涨股最新价 | `"64.3000"` |
| `ts_changeratio` | float | 领涨股涨跌幅 | `0.174858` |
| `ts_ratioamount` | float | 领涨股净流入率 | `0.170321` |

### 注意事项
- ⚠️ **不支持历史数据查询**：虽然接口接受日期参数，但返回的都是最新数据
- ⚠️ **无时间字段**：返回数据中不包含板块更新时间字段
- ⚠️ **实时数据**：每次请求返回当前时刻的排名数据

### Python实现示例

```python
import requests
import json

def get_concept_money_rank(fenlei="1", sort="netamount", page=1, num=20):
    """
    获取概念板块资金排名
    
    Args:
        fenlei: 分类，0=行业，1=概念，2=证监会行业
        sort: 排序字段，netamount=净流入额
        page: 页码
        num: 每页数量
    
    Returns:
        list: 板块资金排名数据列表
    """
    url = "https://vip.stock.finance.sina.com.cn/quotes_service/api/json_v2.php/MoneyFlow.ssl_bkzj_bk"
    
    params = {
        "page": page,
        "num": num,
        "sort": sort,
        "asc": 0,  # 0=降序，1=升序
        "fenlei": fenlei
    }
    
    headers = {
        "Host": "vip.stock.finance.sina.com.cn",
        "Referer": "https://finance.sina.com.cn",
        "User-Agent": "Mozilla/5.0 (Windows NT 10.0; Win64; x64) AppleWebKit/537.36"
    }
    
    try:
        response = requests.get(url, params=params, headers=headers, timeout=10)
        response.raise_for_status()
        return response.json()
    except Exception as e:
        print(f"获取板块资金排名失败: {e}")
        return []

# 使用示例
# 获取概念板块资金排名（按净流入额降序）
concept_ranks = get_concept_money_rank(fenlei="1", sort="netamount", page=1, num=20)
for rank in concept_ranks:
    print(f"{rank['name']}: 净流入 {float(rank['netamount'])/10000:.2f}万元")
```

## 概念板块成分股接口

### 接口信息
- **接口名称**: 板块成分股列表
- **接口路径**: `Market_Center.getHQNodeData`
- **完整URL**: `https://vip.stock.finance.sina.com.cn/quotes_service/api/json_v2.php/Market_Center.getHQNodeData`

### 请求参数

| 参数名 | 类型 | 必填 | 说明 | 示例值 |
|-------|------|------|------|--------|
| `node` | string | 是 | 板块代码（概念板块代码，如gn_ldc） | `"gn_ldc"` |
| `page` | int | 是 | 页码，从1开始 | `1` |
| `num` | int | 是 | 每页数量，建议100-200 | `100` |
| `sort` | string | 是 | 排序字段 | `symbol` |
| `asc` | int | 是 | 排序方向，0=降序，1=升序 | `1` |
| `symbol` | string | 否 | 股票代码过滤（留空获取全部） | `""` |
| `_s_r_a` | string | 否 | 请求来源标识 | `"page"` |

### 排序字段说明（sort参数）

| 值 | 说明 |
|---|------|
| `symbol` | 股票代码（默认） |
| `changepercent` | 涨跌幅 |
| `volume` | 成交量 |
| `amount` | 成交额 |
| `trade` | 最新价 |

### 返回字段说明

返回数据为数组格式，每个元素包含以下字段：

| 字段名 | 类型 | 说明 | 示例值 |
|-------|------|------|--------|
| `symbol` | string | 股票代码（带市场前缀） | `"sh600067"` |
| `code` | string | 股票代码（6位数字） | `"600067"` |
| `name` | string | 股票名称 | `"冠城新材"` |
| `trade` | string | 最新价 | `"4.870"` |
| `pricechange` | float | 涨跌额 | `0.17` |
| `changepercent` | float | 涨跌幅（%） | `3.617` |
| `buy` | string | 买一价 | `"4.870"` |
| `sell` | string | 卖一价 | `"4.880"` |
| `settlement` | string | 昨收价 | `"4.700"` |
| `open` | string | 开盘价 | `"4.690"` |
| `high` | string | 最高价 | `"5.030"` |
| `low` | string | 最低价 | `"4.650"` |
| `volume` | int | 成交量（手） | `156115312` |
| `amount` | int | 成交额（元） | `762032285` |
| `ticktime` | string | 最后交易时间（HH:MM:SS） | `"15:00:01"` |
| `per` | float | 市盈率 | `-9.939` |
| `pb` | float | 市净率 | `1.205` |
| `mktcap` | float | 总市值（万元） | `677742.675893` |
| `nmc` | float | 流通市值（万元） | `677742.675893` |
| `turnoverratio` | float | 换手率（%） | `11.21785` |

### ticktime字段说明
- **格式**: `HH:MM:SS`（如 `"15:00:01"`）
- **含义**: 股票最后交易时间（最后成交时间）
- **特点**: 
  - 不同股票的ticktime可能不同
  - 在多次请求中保持一致（数据快照时间）
  - 通常为收盘时间（15:00）或开盘前时间（09:00）
- **用途**: 
  - 判断数据是否为最新交易日的数据
  - 判断股票是否在交易中
  - 作为数据快照时间的参考
- **限制**: 
  - 不能用于判断板块成分股何时更新
  - 不能用于判断板块信息何时更新

### 注意事项
- ⚠️ **分页限制**: 单次最多返回200条数据，如需获取全部成分股需要分页请求
- ⚠️ **数据快照**: ticktime字段表示数据快照时间，不是实时更新时间
- ⚠️ **无板块更新时间**: 接口不提供板块成分股更新时间字段

### Python实现示例

```python
import requests
import json

def get_concept_stocks(concept_code, page=1, num=200, sort="symbol", asc=1):
    """
    获取概念板块成分股列表
    
    Args:
        concept_code: 板块代码（如 gn_ldc）
        page: 页码
        num: 每页数量（最多200）
        sort: 排序字段
        asc: 排序方向，0=降序，1=升序
    
    Returns:
        list: 成分股数据列表
    """
    url = "https://vip.stock.finance.sina.com.cn/quotes_service/api/json_v2.php/Market_Center.getHQNodeData"
    
    params = {
        "node": concept_code,
        "page": page,
        "num": num,
        "sort": sort,
        "asc": asc,
        "symbol": "",
        "_s_r_a": "page"
    }
    
    headers = {
        "Host": "vip.stock.finance.sina.com.cn",
        "Referer": "https://finance.sina.com.cn",
        "User-Agent": "Mozilla/5.0 (Windows NT 10.0; Win64; x64) AppleWebKit/537.36"
    }
    
    try:
        response = requests.get(url, params=params, headers=headers, timeout=10)
        response.raise_for_status()
        return response.json()
    except Exception as e:
        print(f"获取概念板块成分股失败: {e}")
        return []

def get_all_concept_stocks(concept_code):
    """
    获取概念板块所有成分股（自动分页）
    
    Args:
        concept_code: 板块代码
    
    Returns:
        list: 所有成分股数据列表
    """
    all_stocks = []
    page = 1
    num = 200
    
    while True:
        stocks = get_concept_stocks(concept_code, page=page, num=num)
        if not stocks:
            break
        
        all_stocks.extend(stocks)
        
        # 如果返回的数据少于num，说明已经是最后一页
        if len(stocks) < num:
            break
        
        page += 1
    
    return all_stocks

# 使用示例
# 获取锂电池概念板块成分股
stocks = get_concept_stocks("gn_ldc", page=1, num=100)
print(f"获取到 {len(stocks)} 只成分股")

for stock in stocks[:5]:  # 显示前5只
    print(f"{stock['name']} ({stock['code']}): {stock['trade']}元, 涨跌幅{stock['changepercent']}%")

# 获取所有成分股
all_stocks = get_all_concept_stocks("gn_ldc")
print(f"锂电池概念板块共有 {len(all_stocks)} 只成分股")
```

## 使用限制和注意事项

### 1. 请求频率限制
- ⚠️ **建议请求间隔**: 至少0.5秒，避免请求过快
- ⚠️ **并发限制**: 不建议同时发起大量并发请求
- ⚠️ **超时设置**: 建议设置10秒超时

### 2. 数据更新频率
- **板块资金排名**: 实时数据，每次请求返回最新数据
- **成分股列表**: 数据快照，ticktime字段表示快照时间
- **成分股变化**: 板块成分股变化频率较低，可能需要较长时间才能检测到变化

### 3. 数据限制
- **分页限制**: 成分股接口单次最多返回200条数据
- **历史数据**: 不支持历史数据查询，所有接口返回的都是最新数据
- **时间字段**: 不提供板块创建时间、更新时间等元数据

### 4. 错误处理
- **网络错误**: 建议使用try-except捕获异常
- **超时处理**: 设置合理的超时时间
- **数据验证**: 检查返回数据的格式和完整性

### 5. 板块代码说明
概念板块代码格式：`gn_` + 拼音缩写
- 锂电池: `gn_ldc`
- 锂矿: `gn_lk`
- 股权激励: `gn_gqjl`
- 融资融券: `gn_rzrq`
- 固态电池: `gn_gtdc`

可以通过资金排名接口获取所有板块代码：
```python
# 获取所有概念板块代码
ranks = get_concept_money_rank(fenlei="1", num=1000)
concept_codes = {rank['category']: rank['name'] for rank in ranks}
```

## Python实现示例

### 完整示例类

```python
import requests
import json
import time
from typing import List, Dict, Optional

class SinaConceptAPI:
    """新浪财经概念板块数据API封装类"""
    
    def __init__(self):
        self.base_url = "https://vip.stock.finance.sina.com.cn/quotes_service/api/json_v2.php"
        self.headers = {
            "Host": "vip.stock.finance.sina.com.cn",
            "Referer": "https://finance.sina.com.cn",
            "User-Agent": "Mozilla/5.0 (Windows NT 10.0; Win64; x64) AppleWebKit/537.36 (KHTML, like Gecko) Chrome/117.0.0.0 Safari/537.36 Edg/117.0.2045.60"
        }
        self.timeout = 10
    
    def get_money_rank(self, fenlei: str = "1", sort: str = "netamount", 
                      page: int = 1, num: int = 20) -> List[Dict]:
        """
        获取板块资金排名
        
        Args:
            fenlei: 分类，0=行业，1=概念，2=证监会行业
            sort: 排序字段
            page: 页码
            num: 每页数量
        
        Returns:
            板块资金排名数据列表
        """
        url = f"{self.base_url}/MoneyFlow.ssl_bkzj_bk"
        params = {
            "page": page,
            "num": num,
            "sort": sort,
            "asc": 0,
            "fenlei": fenlei
        }
        
        try:
            response = requests.get(url, params=params, headers=self.headers, 
                                  timeout=self.timeout)
            response.raise_for_status()
            return response.json()
        except Exception as e:
            print(f"获取板块资金排名失败: {e}")
            return []
    
    def get_concept_stocks(self, concept_code: str, page: int = 1, 
                          num: int = 200, sort: str = "symbol", 
                          asc: int = 1) -> List[Dict]:
        """
        获取概念板块成分股
        
        Args:
            concept_code: 板块代码
            page: 页码
            num: 每页数量（最多200）
            sort: 排序字段
            asc: 排序方向，0=降序，1=升序
        
        Returns:
            成分股数据列表
        """
        url = f"{self.base_url}/Market_Center.getHQNodeData"
        params = {
            "node": concept_code,
            "page": page,
            "num": num,
            "sort": sort,
            "asc": asc,
            "symbol": "",
            "_s_r_a": "page"
        }
        
        try:
            response = requests.get(url, params=params, headers=self.headers, 
                                  timeout=self.timeout)
            response.raise_for_status()
            return response.json()
        except Exception as e:
            print(f"获取概念板块成分股失败: {e}")
            return []
    
    def get_all_concept_stocks(self, concept_code: str) -> List[Dict]:
        """
        获取概念板块所有成分股（自动分页）
        
        Args:
            concept_code: 板块代码
        
        Returns:
            所有成分股数据列表
        """
        all_stocks = []
        page = 1
        num = 200
        
        while True:
            stocks = self.get_concept_stocks(concept_code, page=page, num=num)
            if not stocks:
                break
            
            all_stocks.extend(stocks)
            
            if len(stocks) < num:
                break
            
            page += 1
            time.sleep(0.5)  # 避免请求过快
        
        return all_stocks
    
    def get_stock_count(self, concept_code: str) -> int:
        """
        获取板块股票数量
        
        Args:
            concept_code: 板块代码
        
        Returns:
            股票数量
        """
        url = f"{self.base_url}/Market_Center.getHQNodeStockCount"
        params = {"node": concept_code}
        
        try:
            response = requests.get(url, params=params, headers=self.headers, 
                                  timeout=self.timeout)
            response.raise_for_status()
            count = response.json()
            return int(count) if isinstance(count, (str, int)) else 0
        except Exception as e:
            print(f"获取板块股票数量失败: {e}")
            return 0
    
    def get_concept_info(self, concept_code: str) -> Optional[Dict]:
        """
        获取概念板块完整信息（包括资金流向和成分股数量）
        
        Args:
            concept_code: 板块代码
        
        Returns:
            板块信息字典
        """
        # 获取资金流向信息
        ranks = self.get_money_rank(fenlei="1", num=1000)
        money_info = next((r for r in ranks if r.get('category') == concept_code), None)
        
        if not money_info:
            return None
        
        # 获取股票数量
        stock_count = self.get_stock_count(concept_code)
        
        return {
            "code": concept_code,
            "name": money_info.get('name'),
            "stock_count": stock_count,
            "avg_price": money_info.get('avg_price'),
            "avg_changeratio": money_info.get('avg_changeratio'),
            "netamount": money_info.get('netamount'),
            "inamount": money_info.get('inamount'),
            "outamount": money_info.get('outamount'),
            "ratioamount": money_info.get('ratioamount'),
            "leader_stock": {
                "code": money_info.get('ts_symbol'),
                "name": money_info.get('ts_name'),
                "price": money_info.get('ts_trade'),
                "changeratio": money_info.get('ts_changeratio')
            }
        }
    
    def get_latest_ticktime(self, concept_code: str) -> Optional[str]:
        """
        获取板块成分股的最新ticktime（作为数据更新时间参考）
        
        Args:
            concept_code: 板块代码
        
        Returns:
            最新的ticktime值
        """
        stocks = self.get_concept_stocks(concept_code, num=200)
        if not stocks:
            return None
        
        ticktimes = [s.get('ticktime') for s in stocks if s.get('ticktime')]
        return max(ticktimes) if ticktimes else None
```

```python
def safe_get_concept_stocks(concept_code: str, max_retries: int = 3) -> List[Dict]:
    """
    安全获取概念板块成分股（带重试机制）
    
    Args:
        concept_code: 板块代码
        max_retries: 最大重试次数
    
    Returns:
        成分股数据列表
    """
    api = SinaConceptAPI()
    
    for attempt in range(max_retries):
        try:
            stocks = api.get_concept_stocks(concept_code)
            if stocks:
                return stocks
        except requests.exceptions.Timeout:
            print(f"请求超时，第{attempt+1}次重试...")
            time.sleep(2)
        except requests.exceptions.RequestException as e:
            print(f"请求失败: {e}，第{attempt+1}次重试...")
            time.sleep(2)
        except Exception as e:
            print(f"未知错误: {e}")
            break
    
    return []
```

### 数据缓存示例

```python
from functools import lru_cache
import time

class CachedSinaConceptAPI(SinaConceptAPI):
    """带缓存的新浪财经概念板块API"""
    
    def __init__(self, cache_ttl: int = 300):
        """
        Args:
            cache_ttl: 缓存时间（秒），默认5分钟
        """
        super().__init__()
        self.cache_ttl = cache_ttl
        self._cache = {}
        self._cache_time = {}
    
    def _is_cache_valid(self, key: str) -> bool:
        """检查缓存是否有效"""
        if key not in self._cache_time:
            return False
        return time.time() - self._cache_time[key] < self.cache_ttl
    
    def get_money_rank(self, fenlei: str = "1", sort: str = "netamount", 
                      page: int = 1, num: int = 20) -> List[Dict]:
        """带缓存的获取板块资金排名"""
        cache_key = f"money_rank_{fenlei}_{sort}_{page}_{num}"
        
        if self._is_cache_valid(cache_key):
            return self._cache[cache_key]
        
        data = super().get_money_rank(fenlei, sort, page, num)
        self._cache[cache_key] = data
        self._cache_time[cache_key] = time.time()
        
        return data
    
    def get_concept_stocks(self, concept_code: str, page: int = 1, 
                          num: int = 200, sort: str = "symbol", 
                          asc: int = 1) -> List[Dict]:
        """带缓存的获取概念板块成分股"""
        cache_key = f"stocks_{concept_code}_{page}_{num}_{sort}_{asc}"
        
        if self._is_cache_valid(cache_key):
            return self._cache[cache_key]
        
        data = super().get_concept_stocks(concept_code, page, num, sort, asc)
        self._cache[cache_key] = data
        self._cache_time[cache_key] = time.time()
        
        return data
```

## 总结

### 接口特点
1. ✅ **数据实时性**: 所有接口返回的都是最新数据
2. ✅ **数据完整性**: 提供详细的板块和股票信息
3. ⚠️ **历史数据**: 不支持历史数据查询
4. ⚠️ **时间字段**: 不提供板块创建时间、更新时间等元数据

### 推荐使用方式
1. **获取板块列表**: 使用资金排名接口获取所有板块代码和名称
2. **获取成分股**: 使用成分股接口获取板块内所有股票
3. **数据更新检测**: 通过定期对比成分股列表检测变化
4. **时间参考**: 使用ticktime字段的最大值作为数据新鲜度参考

### 注意事项
- 合理控制请求频率，避免对服务器造成压力
- 处理网络异常和超时情况
- 验证返回数据的格式和完整性
- 如需历史数据，需要自己定期抓取并存储
